# Learning React

A simple React practice project featuring dynamic person components with toggle functionality.
This is based on the small example from https://react.dev/learn.

## Features
- Dynamic component rendering with map()
- useState hook for show/hide descriptions  
- CSS styling and flexbox layout
- Click events and conditional rendering

## What I learned
- React component structure and props
- State management with useState
- Event handling in React
- CSS-in-JS styling approaches
